9.1. Internal Variables

第九章第一節 - 內部變數
---

Builtin variables:
>`內建參數：`

variables affecting bash script behavior
>`影響到 bash script 行為者，稱內建參數。`

$BASH

The path to the Bash binary itself
>`Bash binary 檔本身路徑。`

```bash
bash$ echo $BASH
/bin/bash
```
